# 💡 Why I Chose Vanilla JavaScript

Instead of using frameworks like React, Next.js, or Vue, I intentionally built this portfolio using only **HTML, CSS, and JavaScript**.

## ⚡ Speed

* No build tools
* No npm packages
* No unnecessary dependencies
* Extremely lightweight
* Instant loading experience

---

## 🧠 Simplicity

Everything is easy to understand.

Anyone can open the project and immediately know how it works without learning a framework or complex tooling.

---

## 🔧 Easy Maintenance

The project contains only three core files.

* HTML for structure
* CSS for styling
* JavaScript for interactivity

No package updates.
No breaking framework changes.
No configuration headaches.

---

## 🌍 Future Proof

The web is built on HTML, CSS, and JavaScript.

This portfolio will continue working years from now without depending on framework updates or changing ecosystems.

---

## 🎯 Right Tool for the Job

A portfolio website mainly contains:

* Information
* Projects
* Contact links
* Small animations

It doesn't require complex state management or hundreds of reusable components.

Using a large framework here would simply increase complexity without improving the user experience.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

Move into the project

```bash
cd portfolio
```

Open the project

```bash
open index.html
```

or start a local server

```bash
python3 -m http.server 8000
```

Visit

```
http://localhost:8000
```

---

## Modify Colors

Inside **style.css**

```css
:root{
    --primary:#FFD700;
    --secondary:#00D4FF;
    --accent:#FF4D6D;
}
```
